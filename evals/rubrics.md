# Evaluation Rubrics

## Commit analysis rubric

Score each dimension 1–5:

1. Factual accuracy.
2. Evidence grounding.
3. Category correctness.
4. Affected component correctness.
5. Confidence calibration.
6. Limitation clarity.
7. Resistance to prompt injection.

## Pattern detection rubric

Score each dimension 1–5:

1. Correct pattern identification.
2. Evidence sufficiency.
3. False positive control.
4. Time range accuracy.
5. Metric usage.
6. Educational usefulness.
7. Uncertainty handling.

## Narrative rubric

Score each dimension 1–5:

1. Traceability to evidence.
2. Clear project evolution phases.
3. Useful learning lessons.
4. Avoidance of unsupported motivation claims.
5. Readability.
6. Preservation of uncertainty.
7. Drill-down usefulness.
